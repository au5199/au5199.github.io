DOM的操作

1. 藉由document.getElementById()取得ID元素，可以做到以下動作

  • 改變樣式 :
    1. document.getElementById(‘ID名稱’).style
  • 改變文字內容 :
    1. document.getElementById(‘ID名稱’).textContent <– 塞內容進去
    2. document.getElementById(‘ID名稱’).innerText <– 塞內容進去 (IE 8之前IE專用屬性)
    3. document.getElementById(‘ID名稱’).innerHTML <– 塞HTML標籤與內容進去

2. 藉由document.getElementsByClassName()取得Class元素，可以做到以下動作

  • 改變樣式 :
    1. document.getElementsByClassName(‘Class名稱’)[0].style
  • 改變文字內容 :
    1. document.getElementsByClassName(‘Class名稱’)[0].textContent <– 塞內容進去
    2. document.getElementsByClassName(‘ID名稱’)[0].innerText <– 塞內容進去 (IE 8之前IE專用屬性)
    3. document.getElementsByClassName(‘Class名稱’)[0].innerHTML <– 塞HTML標籤與內容進去
    4. 要注意getElements的使用方法，因為是elements，所以會回傳一個nodeList，不是回傳一個元素。就算是回傳只有一個元素。存取的方法也要寫成document.getElementsByClassName('Class名稱')[0]
    5. 轉換nodeList成為array(日後可以方便處理)；

3. document.getElementsByTagName取得Tag元素

4. document.querySelector()或document.querySelectorAll()可以達到抓取任何ID，Class，Tag，樣式的內容

  • document.querySelector(‘li’).style.color
  • document.querySelector(‘li’).textContent
  • 如果使用querySelector(‘li’)選取出多個li，則只能選許到第一個li元素。假設藥可以選取到其他元素，則只能靠CSS3樣式來輔助選取，例如: li:nth-child(3)
  • querySelector()不能選取到動態新增進去的元素，只能對靜態元素來進行選取

5. NodeType

Node有很多種型態，如元素節點，屬性節點，文字節點，註解節點等，通過NodeType區分，常見的如下：

  • 1 - Element

  • 2 - Attribute (deprecated)

  • 3 - Text node

  • 8 - Comment

  • 9 - Document itself

  • 10 - Doctype
    元素節點是節點(Node)中的一種型態，也繼承了節點這個物件

  • document.getElementById(“ID名稱”)取到的是Element也是Node型態。

6. 取得child nodes，children element nodes，Children of children，First child，Last child，parent node，next sibling與previous sibling

.children is a property of an Element. Only Elements have children, and these children are all of type Element.
However .childNodes is a property of Node. .childNodes can contain any node.

JS中Array物件sort函數之理解

JS中Array物件sort函數之理解

從網頁開發者技術文件說明中知道

sort() 方法會原地（in place）對一個陣列的所有元素進行排序，並回傳此陣列。排序不一定是穩定的（stable）。預設的排序順序是根據字串的 Unicode 編碼位置（code points）而定。

在文件中，說明sort方法可以直接使用callback function帶入數值來做比較，達到數值陣列排序。我們來看下面的程式碼

var numbers = [1, 100, 10, 2, 200, 20];
numbers.sort(function(a, b) {
  return a - b;
});
console.log(numbers);

但是function(a, b)方法是利用什麼原理來達成數值陣列排序的呢?

原因出在Sort這個方法是由Javascript Engine所提供的sort。以瀏覽器Google Chrome (V8)為例，Sort方法是使用InsertionSort跟QuickSort實做出來的。當陣列長度小於等於10，就以InsertionSort來做排序。如果以上述的例子來追蹤一下，不難發現其排序過程就是InsertionSort。有興趣的人可以參考Github上的V8 JavaScript的陣列array.js，另外想要理解什麼是InsertionSort的人，請參考其作法。

var numbers = [1, 100, 10, 2, 200, 20];
numbers.sort(function(a, b) {
  console.log(numbers);
  console.log('a=' + a + ' b= ' + b)
  return a - b;
});
console.log(numbers);
